# между

> третий, кто слушает обоих

AI-собеседник для разговоров о чувствах, ссорах и отношениях. Не терапевт. Эмоциональный переводчик и зеркало паттернов.

Построено на Next.js 14 + Claude API (Sonnet 4.6).

---

## 🚀 Быстрый старт (локально)

### 1. Получи API-ключ от Anthropic

- Зайди на https://console.anthropic.com/
- Зарегистрируйся, добавь немного денег ($5-10 хватит надолго для теста)
- Создай API-ключ в разделе **API Keys**
- Скопируй ключ (он начинается с `sk-ant-api03-...`)

### 2. Установи Node.js

Если ещё не установлен — https://nodejs.org/ (LTS-версия).

Проверь в терминале:
```bash
node --version  # должно показать v18+ или v20+
```

### 3. Установи зависимости

В терминале, в папке проекта:
```bash
npm install
```

### 4. Создай файл `.env.local`

В корне проекта создай файл с именем `.env.local` (точно так, с точкой в начале) и положи туда:

```
ANTHROPIC_API_KEY=sk-ant-api03-твой_ключ
```

⚠️ **Никогда не коммить этот файл в git** — он уже в `.gitignore`.

### 5. Запусти

```bash
npm run dev
```

Открой http://localhost:3000 — должно работать.

## 🌍 Деплой на Vercel (бесплатно)

Это самый простой способ выложить онлайн.

### Вариант A: через GitHub (рекомендую)

1. Создай аккаунт на https://github.com/ если нет
2. Создай новый репозиторий (например `mezhdu`)
3. В терминале в папке проекта:
   ```bash
   git init
   git add .
   git commit -m "first commit"
   git remote add origin https://github.com/твой-логин/mezhdu.git
   git push -u origin main
   ```
4. Зайди на https://vercel.com, войди через GitHub
5. Нажми **Add New → Project**, выбери репозиторий `mezhdu`
6. В разделе **Environment Variables** добавь:
   - Name: `ANTHROPIC_API_KEY`
   - Value: твой ключ
7. Нажми **Deploy**
8. Через ~1 минуту получишь URL типа `mezhdu-xyz.vercel.app`

### Вариант B: через Vercel CLI

```bash
npm i -g vercel
vercel
```

Следуй подсказкам, потом добавь переменную окружения через Vercel-дашборд.

## 📁 Структура проекта

```
mezhdu/
├── app/
│   ├── layout.jsx         # обёртка html/body
│   ├── page.jsx           # ВСЯ ЛОГИКА UI — здесь править внешний вид
│   └── api/
│       └── chat/
│           └── route.js   # серверный вызов Claude — здесь править системный промпт
├── package.json
├── next.config.js
├── .env.local             # твой ключ (создаёшь сам, в git не попадает)
└── .gitignore
```

### Что где править

| Хочу изменить                  | Где                                       |
|--------------------------------|-------------------------------------------|
| Системный промпт (тон AI)      | `app/api/chat/route.js` → `SYSTEM_PROMPT` |
| Тексты на странице             | `app/page.jsx` → `STRINGS`                |
| Цвета/темы                     | `app/page.jsx` → `THEMES`                 |
| Стартовые карточки             | `app/page.jsx` → `STRINGS.ru.starters`    |
| Модель Claude                  | `app/api/chat/route.js` → `model:`        |

## 🔑 Безопасность

- API-ключ лежит **только на сервере** (`.env.local`, Vercel env vars). В браузер не утекает.
- Все вызовы к Claude идут через `/api/chat` — это твой роут.
- Если хочешь добавить лимиты по IP, чтобы не сжечь весь баланс — добавь rate-limiting middleware (например `@upstash/ratelimit`).

---

## ⚠️ Важные замечания

- **Микрофон** работает в Chrome, Safari (iOS 14.5+), Edge. В Firefox — не работает (там Web Speech API не поддерживается). На сайте, развёрнутом на Vercel (HTTPS), микрофон будет работать нормально — в отличие от артефакта в iframe.
- **Без памяти**: разговоры пока не сохраняются между сессиями. Это первая фича, которую стоит добавить (см. ниже).
- **Без авторизации**: пока любой с URL может пользоваться и тратить твои токены. Добавь auth (NextAuth/Clerk), когда захочешь сделать публичным.

## 💰 Сколько стоит запустить

Claude Sonnet 4.6: ~$3 за миллион входных токенов, ~$15 за миллион выходных.
Один разговор из ~20 сообщений стоит примерно $0.05–$0.15.
$10 на балансе Anthropic ≈ 100-200 содержательных разговоров.
Vercel free tier — бесплатно для личного использования и небольшого трафика.
